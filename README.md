# Frontend

## Lưu ý

- `eslint ` sẽ chạy với mỗi commit, nên hãy fix error cho các file được thay đổi trước rồi hãy commit nhé

## Commands

- development: `npm run dev`

- build: `npm run build`

## Import alias

tham khảo app.jsx và index.jsx

- `@/styles` -> `src/styles`

- `@/components` -> `src/components`

- `@/pages` -> `src/pages`

- `@/` -> `./`

- `@/public` -> `public`
